# react-native-sms-user-consent

React Native SMS User Consent

## Installation

```sh
npm install react-native-sms-user-consent
```

## Usage

```js
import SmsUserConsent from "react-native-sms-user-consent";

// ...

const result = await SmsUserConsent.multiply(3, 7);
```
